# SNS RULES (X投稿文)

## 基本方針
- X投稿文は「面白おかしいけど怪しくない」文体を基本にする。
- 真面目すぎる説明文はタイムラインで流れやすい。
- 1行目はクスッとするフックにする。
- 2〜3行目で何が確認できるかを自然に説明する。
- 医療・保険・お金系は断定しない。
- 最後は「無料チェック👇」+ URL で統一する。

## 禁止ワード / NG表現
- 「絶対」「必ず」「保証」「儲かる」「誰でも稼げる」は使わない。
- 過剰に不安を煽らない。
- 下品・攻撃的・差別的な表現を使わない。

## フック例
- 退去費用、最後のボス感ある。
- 一人暮らし、財布のHP削られがち。
- 住民税、忘れた頃に現れるラスボス。
- 給料、入った瞬間どこかへ旅立つ説。
- 防災グッズ、買おうと思って3年経ってる。

## 追加ルール（2026-05-01 追記）
- X投稿文は長く説明しすぎない（短く読める構成を優先）。
- 1行目は「ちょい面白いフック」にする。
- 2〜3行目は短い疑問文で引っかける。
- 4行目で行動を促す。
- 注意書きは短く自然に入れる。
- 最後は URL で締める。
- 「無料チェック👇」は使ってもよいが、毎回固定で入れすぎない（広告感を避けるため文面に合わせて調整）。
- 断定や過剰煽りは避ける。

## 採用例
退去費用、最後にドンと来がち。

敷金で足りる？
原状回復ってどこまで見る？

あとで焦る前に、
ざっくり目安を無料チェック。
※断定ではなく、確認用です
