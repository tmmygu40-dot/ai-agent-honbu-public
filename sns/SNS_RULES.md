# SNS RULES (X投稿文)

## 基本方針
- X投稿文は「面白おかしいけど怪しくない」文体を基本にする。
- 真面目すぎる説明文はタイムラインで流れやすい。
- 1行目はクスッとするフックにする。
- 2〜3行目で何が確認できるかを自然に説明する。
- 医療・保険・お金系は断定しない。
- 最後は「無料チェック👇」+ URL で統一する。

## 禁止ワード / NG表現
- 「絶対」「必ず」「保証」「儲かる」「誰でも稼げる」は使わない。
- 過剰に不安を煽らない。
- 下品・攻撃的・差別的な表現を使わない。

## フック例
- 退去費用、最後のボス感ある。
- 一人暮らし、財布のHP削られがち。
- 住民税、忘れた頃に現れるラスボス。
- 給料、入った瞬間どこかへ旅立つ説。
- 防災グッズ、買おうと思って3年経ってる。
