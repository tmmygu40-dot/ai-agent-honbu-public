## 2026-04-30 本日追加バッジが朝に消える問題（JST対応）

### 症状
- 「本日◯件追加」バッジが毎日 0:00〜9:00（JST）に表示されない

### 原因
- 日付判定に `toISOString()`（UTC）を使用していたため、日本時間とズレが発生

### 対応
- JSTローカル日付で比較するように修正
  ```js
  var n = new Date();
  var today = n.getFullYear() + '-' +
              String(n.getMonth() + 1).padStart(2, '0') + '-' +
              String(n.getDate()).padStart(2, '0');
  ```

## 2026-05-01 X投稿文体ルール更新（学習メモ）

### 方針
- X投稿文は「面白おかしいけど怪しくない」文体を基本にする
- 真面目すぎる説明文はタイムラインで流れやすいため、1行目で目に止まるフックを優先
- 2〜3行目で「何が確認できるか」を自然に説明する
- 医療・保険・お金系は断定しない（判断材料ベースで表現）
- 文末は「無料チェック👇」+ URL で統一する

### 禁止ワード / NG方向
- 「絶対」「必ず」「保証」「儲かる」「誰でも稼げる」は使わない
- 過剰な不安煽りはしない
- 下品・攻撃的・差別的な表現はしない

### フック例
- 退去費用、最後のボス感ある。
- 一人暮らし、財布のHP削られがち。
- 住民税、忘れた頃に現れるラスボス。
- 給料、入った瞬間どこかへ旅立つ説。
- 防災グッズ、買おうと思って3年経ってる。

### 追記（短文化ルール）
- 説明は長くしすぎず、1行目フック→2〜3行目短い疑問文→4行目行動喚起の流れを優先
- 注意書きは短く自然に入れる
- 「無料チェック👇」は毎回固定にせず、広告感が強くなりすぎないよう文面に合わせて調整
- 断定・過剰煽りは引き続き禁止

### 追記（先頭空行ルール）
- X表示上の視認性を考慮し、投稿文先頭に空行を1つ入れる
- 1行目は空行、2行目からフック開始
- 構成は「空行→フック→短いリズム文→説明2行→注意書き→URL」を基本形とする

### 追記（1行目ラベル方式へ変更）
- 先頭空行はX側の表示で詰まって見えるため廃止
- 1行目を「【今日のチェック】」固定に変更
- 2行目を空行、3行目からフック開始に統一
