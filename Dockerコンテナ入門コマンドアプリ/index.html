Docker入門コマンドナビ

初心者向け基本6コマンドを目的別にサクッと検索

「コンテナを起動したい」「ログを確認したい」など、やりたいことを選ぶだけで対応するDockerコマンドと実行例が表示される学習ツール。初心者の最初の一歩をお手伝いします。

使い方

  1. 目的を選択：下のドロップダウンからやりたいことを選んでください
  2. コマンド表示：「コマンドを表示」ボタンで対応コマンドと説明が出ます
  3. 実行例を確認：コマンドの使い方・オプション・関連コマンドを参考に実行してみましょう

よくある目的から選ぶ

あなたの目的を入力

目的を選択して「コマンドを表示」をクリックするとコマンドが表示されます

このコマンドについて

  • 役割：Docker コンテナの基本操作を実現するコマンド。初心者が最初に覚えるべき重要な機能です。
  • オプション：コマンドに -d（バックグラウンド実行）・-it（対話モード）・-p（ポート指定）などのオプションを組み合わせることで動作をカスタマイズできます。
  • 関連コマンド：このコマンドを覚えたら docker inspect・docker exec・docker build なども合わせて学習すると Docker スキルが格段に上達します。

よくある質問

Q. このツールの計算内容やコマンド情報は外部に送信されますか？
A. 送信しません。すべてブラウザ内で完結します。あなたの選択は完全に手元で処理されます。
Q. Docker がインストールされていなくてもこのアプリは使えますか？
A. はい。このアプリはコマンドを学ぶためのガイドなので、Docker がなくても確認・学習できます。実際に実行するときに Docker をインストールしてください。
Q. 複数オプションを組み合わせたコマンドを作りたい場合は？
A. このアプリは基本コマンド 6 種類の最頻用例を収載しています。複雑な組み合わせや詳細な挙動は公式ドキュメント（Docker Docs）で確認するのがおすすめです。
Q. Windows・Mac・Linux で動作しますか？
A. Docker 自体は 3 つのプラットフォーム全てで動作します。このアプリで学んだコマンド文法は基本的に共通ですが、細かいシステムパス設定などは環境に応じて異なる場合があります。
