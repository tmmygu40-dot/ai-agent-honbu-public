二次方程式をサクッと解く！

ax² + bx + c = 0 の係数を入力すると、解をすぐに計算します。練習問題の答え合わせや学習に。

  1. a, b, c の係数を入力します
  2. 「計算する」ボタンをクリック
  3. 解と判別式の結果が表示されます

サンプルデータを試す：

計算結果がここに表示されます
係数を入力して「計算する」をクリック
  • 判別式 D = b² - 4ac で、実根・重根・虚根を判定します
  • D ≥ 0 なら実数解が存在し、公式 x = (-b ± √D) / 2a で求まります
  • D < 0 なら虚数解となります
Q. 入力データは外部に送信されますか？
A. 送信しません。すべてブラウザ内で計算が完結します。
Q. 分数や小数で答えが出ますか？
A. 小数で表示されます。分数形式が必要な場合は手動で変換してください。
Q. 虚数解の場合はどう表示されますか？
A. 実部と虚部を分けて a + bi 形式で表示されます。
