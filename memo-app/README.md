# シンプルメモアプリ

開いてすぐ書ける。それだけのメモアプリです。

---

## できること

- メモを書いて保存する
- 書いたメモを一覧で見る
- いらないメモを削除する

---

## 使い方

1. `index.html` をブラウザで開く
2. 入力欄にメモを書く
3. 「追加する」ボタンを押す（または Ctrl + Enter）

---

## 特徴

- インターネット不要（ブラウザだけで動く）
- メモはブラウザに自動保存される（localStorage 使用）
- サーバー不要・アカウント不要
- HTML / CSS / JavaScript だけで作られている

---

## 注意

- メモはこのブラウザのみに保存される
- ブラウザのデータを消すとメモも消える
- 他の端末とは共有されない

---

## 使った技術

- HTML
- CSS
- JavaScript（localStorage）
