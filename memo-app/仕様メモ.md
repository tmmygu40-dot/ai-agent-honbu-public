# 仕様メモ｜シンプルメモアプリ

このアプリがどう動くかをまとめたメモです。

---

## 基本の動き

| やること | どうやるか |
|---|---|
| メモを書く | 入力欄に文字を打つ |
| メモを保存する | 「追加する」ボタン または Ctrl+Enter |
| メモを見る | ページを開くと自動で表示される |
| メモを消す | 各メモの「✕」ボタンを押す |

---

## データの保存方法

- ブラウザの `localStorage` を使う
- キー名：`simple-memo-list`
- 保存形式：JSON（配列）
- 1件のメモに入る情報：
  - `id`（数字・追加した時刻）
  - `text`（メモの本文）
  - `date`（保存した日時・表示用）

## ファイル構成

```

├── .gitignore   # GitHubに上げないファイルの設定
└── 仕様メモ.md  # このファイル（設計メモ）
```
