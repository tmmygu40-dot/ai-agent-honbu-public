# ネコポケ公式マスコット｜画像生成用キャラ設定メモ

**用途**: ChatGPT / BOB などへ画像生成を依頼するときの固定仕様。  
**Claude / Cursor では画像を生成しない**（本ファイルはプロンプト設計・共有用）。

**参考画像**: ブランド側で参照用画像がある場合は、**画像生成ではその見た目を最優先**する（本メモと矛盾するなら参照画像に従う）。

**固有名詞**: 公式サイトに未掲載のキャラクター名・愛称を**新設しない**（呼び出しは「ネコポケのマスコット猫」「案内役の猫型マスコット」など役割表現で足りる）。

---

## 1. 役割

- **ネコポケ（猫型ポケット）の案内役・宣伝役**として使う、**猫型のマスコット**。
- ゆるい親しみやすさ。説教調・威圧感は避ける。

---

## 2. 見た目の固定ルール（必ず守る）

| 部位・要素 | 指定 |
| --- | --- |
| **全体** | 猫型マスコット。**リアル猫・写真風にしない**。イラスト／キャラクター調。 |
| **配色（本体）** | **くすみブルー系**が胴・背面の主色。 |
| **配色（顔まわり・お腹）** | **クリーム色**（顔周りの塊、お腹の塊としてはっきり分かる）。 |
| **お腹のポケット** | **お腹にポケットがある**。**描き忘れ禁止**。 |
| **手** | **指なし**。**丸いミトン型**の手。**指を描かない**。 |
| **しっぽ** | **丸くて赤い**。**描き忘れ禁止**。 |
| **体型** | **ずんぐり丸い**。**2頭身〜2.5頭身**。 |
| **足** | **短く丸い**。**細くしない**。 |
| **表情** | **ゆるく親しみやすい**。 |

### しっぽの補足

- **長い猫しっぽにしない**（丸い・短めのシルエット）。
- **青やブルー系のしっぽにしない**（赤に固定）。

---

## 3. NG事項（生成後チェック用）

- **指ありの手**にしない（ミトン手のみ）。
- **ポケットを消さない**／省略しない。
- **しっぽを長い猫しっぽ**にしない。
- **しっぽを青**にしない（赤・丸）。
- **細身・スレンダー**にしない。
- **リアル猫・リアル毛並み**にしない。
- **公式として未確定の固有名詞**（キャラ名など）をプロンプトやキャプションに**勝手に付けない**。
